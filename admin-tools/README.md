# ابزارهای مدیریتی طب‌نما

این پوشه شامل ابزارهایی برای کمک به مدیریت سایت طب‌نما است.

## نصب و راه‌اندازی

1. اطمینان حاصل کنید که Node.js روی سیستم شما نصب است
2. در این پوشه یک پنجره ترمینال باز کنید
3. دستور زیر را اجرا کنید:

```bash
npm install
```

## استفاده از ابزارها

### ایجاد تجربه جدید

برای ایجاد یک تجربه جدید، می‌توانید از اسکریپت `generate-review.js` استفاده کنید:

1. فایل `generate-review.js` را باز کنید
2. اطلاعات تجربه را در قسمت `sampleReview` وارد کنید
3. خط مربوط به `createReview` را فعال کنید (علامت // را بردارید)
4. دستور زیر را اجرا کنید:

```bash
node generate-review.js
```

یا می‌توانید از دستور npm استفاده کنید:

```bash
npm run generate-review
```

### استفاده به صورت برنامه‌نویسی

شما می‌توانید از توابع موجود در اسکریپت به صورت برنامه‌نویسی نیز استفاده کنید:

```javascript
const { createReview, hashEmail } = require('./generate-review');

const reviewData = {
  email: "user@example.com",
  job_name: "بیمار",
  rate: 4.5,
  state: "نامشخص",
  agent: "Mozilla/5.0 ...",
  description: "توضیحات تجربه...",
  pros: "امتیاز مثبت اول,امتیاز مثبت دوم",
  cons: "امتیاز منفی اول,امتیاز منفی دوم"
};

createReview(reviewData, "نام-کلینیک-یا-پزشک");
```

## نکات مهم

- ایمیل کاربران به صورت خودکار هش می‌شود تا حریم خصوصی آن‌ها حفظ شود
- شناسه‌های یکتا به صورت خودکار تولید می‌شوند
- تاریخ به صورت timestamp ثبت می‌شود
- نام پوشه باید با slug مقاله مربوطه مطابقت داشته باشد
